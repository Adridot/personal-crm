# V2+ Backlog

This backlog is intentionally structured rather than fully issue-ready. These items should be promoted only after MVP and V1 prove stable.

## PWA and Mobile

- Add installable PWA behavior.
- Explore offline-friendly caching and sync rules.
- Add push notifications for reminders when product value justifies them.

## Messaging and Beeper

- Research Beeper API capabilities and constraints.
- Define how messages map to contacts and interaction timelines.
- Decide whether full message content, metadata, or summaries are acceptable.

## AI and Natural-Language Features

- Natural-language reminder creation.
- More advanced follow-up suggestions.
- AI refinement of relationship score models.
- Sentiment analysis on notes or message-derived context if messaging is added.

## Custom Fields System

- Decide storage model for dynamic fields.
- Build typed custom field definitions and validation.
- Design UI for creating, ordering, and displaying user-defined fields.

## Capture and Enrichment

- Browser extension for quick contact capture from web pages.
- OCR business card scanning.
- Contact enrichment via public-profile APIs.
- Optional contextual/news feeds tied to contacts.

## Advanced UX

- Board, gallery, or calendar views.
- Google Calendar quick scheduling for fast meetup planning.
- Saved views and richer filtering.
- More mobile-oriented UI patterns once PWA direction is approved.

## Research Gate

Before any V2+ work starts, re-check:

- provider policy constraints,
- privacy implications,
- operational cost,
- and whether the feature supports the app's core relationship-management value.
