# Roadmap

## Phase 0: Documentation and Repo Bootstrap

Goal:

- Freeze the product and architecture baseline.
- Prepare the repository structure and issue backlog.

Expected outputs:

- `project/` documentation workspace
- approved MVP backlog
- initial repo bootstrap issues

## Phase 1: MVP Foundation

Goal:

- Create the monorepo, frontend shell, backend shell, database setup, and authentication baseline.

Expected outputs:

- workspace tooling
- PostgreSQL + Drizzle setup
- Better Auth integration
- translation-ready frontend shell

## Phase 2: MVP Completion

Goal:

- Deliver the end-to-end user value of importing contacts, managing them, recording interactions, using reminders, and seeing a dashboard.

Expected outputs:

- Google Contacts import
- contacts CRUD
- tags and relationships
- interaction timeline
- reminders
- dashboard
- export and account deletion

## Phase 3: V1 Enrichments

Goal:

- Add the first layer of intelligence and broader contact source support.

Expected outputs:

- Apple/iCloud import
- additional sources
- duplicate merge suggestions
- connection scoring
- contact task presets
- gifts/debts/ideas

## Phase 4: V2+ Research and Expansion

Goal:

- Explore richer mobile, AI, ingestion, and extensibility features without destabilizing the core product.

Expected outputs:

- PWA decisions
- Beeper research
- custom fields design
- capture and enrichment exploration
